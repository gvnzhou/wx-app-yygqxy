## 粤语歌曲谐音

### 前言

一个提供粤语歌曲谐音查询的微信小程序，数据爬取自网络。icon取自阿里的iconFont。

### 项目目录说明

- pages      --------- 所有页面存放目录
- images     --------- 图片资源目录
- utils      --------- 工具函数目录
- components --------- 通用组件目录
